FUNCTIONS

A JavaScript function is a block of code designed to perform a particular task. That JavaScript function is executed when 'something' invokes it, this also can be referenced as 'calling' a function. You define a JavaScript function with the function keyword, and you will then put a name followed by parantheses. Within a function you use return statement to have the function stop executing.

OBJECTS

Objects are technically variables as well, but objects are able to contain many values. Objects have properties which can be name, color, age etc. and then they have values of those properties. The name:values pairs in JavaScript objects are called properties. You can reference a object property by the notation of myCar.name or myCar["name"].

ARRAYS

An array is a special variable, JavaScript arrays are used to store multiple values in a single variable. Arrays are useful for storing a large amount of information and then being able to loop thorugh it to find a specific piece of information. In order to access an array element you refer to it by the index number myArr[3].

FOR LOOPS

Loops can execute a block of code a number of times! This is especially helpful if you want to run the same code over and over again, each time with a different value. JavaScript supports different kinds of loops as well which are 'for', 'for/in', 'for/of', 'while', 'do/while'. You will always need to have 2 semicolons within the parantheses of the for loop in order for it to function correctly.

CONDITIONALS

Conditionals control behavior in JavaScript and determine whether or not pieces of code can run. The most common type of conditional is the if statement, which only runs if the condition enclosed in parentheses is true. You can extend an if statement with an else statement or an if else statement if the first conditional does not pass.
